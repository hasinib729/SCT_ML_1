# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using Linear Regression based on features like:
- Square Footage (GrLivArea)
- Number of Bedrooms
- Number of Bathrooms

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 🚀 Steps Performed
1. Data Loading
2. Data Cleaning
3. Feature Selection
4. Model Training
5. Evaluation (R² Score, MSE)
6. Prediction & Submission

## 📊 Results
- Model trained using Linear Regression
- Evaluated using Mean Squared Error & R² Score

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
```

## 📁 Output
- submission.csv (Predicted house prices)
